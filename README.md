# pomelo
pomelo framework

[專案介紹 pomelo](https://hackmd.io/@chiisen/HJFpzAfoY)

檢查 pomelo 是否安裝成功
```shell=
pomelo --version
```

初始化專案
```shell=
pomelo init
```

安裝套件
```shell=
sh ./npm-install.sh
```
![npm-install.sh](https://i.imgur.com/419WoZo.png)

安裝 redis
```shell=
npm install redis
```
參考: [Node-Redis](https://www.npmjs.com/package/redis)
